# 🛠️ Predictive Maintenance: Прогнозирование поломок двигателей

Данный проект посвящен разработке системы раннего обнаружения отказов оборудования. Мы сравниваем классический машинный алгоритм и глубокое обучение для предсказания вероятности поломки на основе телеметрии датчиков.

## 📌 Описание задачи
Цель — предсказать, выйдет ли двигатель из строя в ближайшее время (бинарная классификация). Проект решает проблему дисбаланса классов и учитывает временную зависимость данных.

---

## 🏗️ Используемые архитектуры

В проекте реализованы и сравнены три подхода:

### 1. XGBoost (Base)
* **Тип:** Градиентный бустинг над деревьями решений.
* **Особенности:** Обработка данных "здесь и сейчас", высокая скорость, компенсация дисбаланса через `scale_pos_weight`.

### 2. LSTM (Deep Learning)
* **Тип:** Рекуррентная нейросеть с памятью.
* **Особенности:** Работает со скользящим окном (**30 циклов истории**). Анализирует динамику износа и тренды датчиков. Построена на PyTorch.


### 3. XGBoost (Optuna Tuned)
* **Тип:** Оптимизированный бустинг.
* **Особенности:** Гиперпараметры подобраны с помощью **Optuna** (TPE Sampler) для максимизации метрики **Recall** (минимизация пропусков поломок).
